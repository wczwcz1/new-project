# Simple Password Generator (简易密码生成器)

这是一个使用 Python 编写的轻量级命令行密码生成器。它可以根据用户指定的长度，快速生成包含大小写字母、数字和特殊符号的高强度随机密码。

## ✨ 功能特点
* **开箱即用**：纯 Python 标准库编写，无需使用 `pip` 安装任何第三方包。
* **高安全性**：自动混合字母、数字和标点符号，防止密码被暴力破解。
* **灵活定制**：支持用户自定义密码长度。
